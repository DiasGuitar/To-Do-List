# To-Do List Web Application

A simple To-Do List application built using the MERN stack.

## Technologies Used

- MongoDB
- Express JS
- React JS
- Node.js
- Nodemon
- CORS
- Axios
- SCSS
- HTML
- CSS
- Mongoose

## Features

- Add tasks to the to-do list.
- Mark tasks as "done" with a checkbox.
- Delete tasks.
- Choose between 5 different color themes.
- Mobile-Responsive
